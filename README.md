# 指定进制下的素数末位数字分布分析程序
> 通过生成素数，统计素数在指定进制下的末位数字分布并可视化结果
## 狄利克雷定理(Dirichlet's theorem on arithmetic progressions)
> 若整数a和b满足gcd(a,d)=1，即a、d互质，则a, a+d, a+2d... 中包含无穷多个素数
进一步地，在解析数论的框架下可以证明：当x→∞ 时，不超过x的素数在模d的每一个与d互质的剩余类中渐近均匀分布，即每个此类剩余类包含约1/φ(d)比例的素数。
## 项目简介
本项目将「进制 base」等价于定理中的「模数 d」，「素数末位数字」等价于「素数对 base 取模的剩余类」，通过统计和可视化验证这一规律：
1. 仅与进制互质的末位数字会出现大量素数（如十进制下仅 1/3/7/9 高频出现）；
1. 随着素数范围扩大，互质末位的素数频率趋近均匀。
## 功能特点
1. 基于 `sympy`生成指定上界内的素数
2. 支持任意大于2的进制，通过取模获取末位
3. 生成有标注的柱状图
4. 自动校验进制、数值合法性
## 快速开始
### 环境要求
1. Python 3.6 及以上版本
2. 需要安装以下库 `pip install sympy matplotlib`
### 安装与运行
1. 运行代码 `python Dirich.py`
2. 输入目标进制(需要大于等于2)
3. 输入上界，默认为1000000，该输入过大会增加运行时间
4. 输出结果：该上界以内的素数个数以及一张展示末位数字分布频率的柱状图(每个柱子均标注了具体数值)
